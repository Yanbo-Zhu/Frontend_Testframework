## 3.2 within 

同时获取多个子元素 

```js
it('within',() => {
    cy.visit('https://example.cypress.io/commands/querying')
    cy.get('.query-form').within( ($form)=> {   // $form 表示 在前面 通过 cy.get('.query-form') 找到的 form 表单 
        cy.get('input:first').type("yuz")
        cy.get('input:last').type("123456")
    })
});

```
